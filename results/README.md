# Results and Evaluation Outputs

This folder contains visualizations and evaluation artifacts
generated from the GRU-based egg price prediction model.

---

## Files Description

### 1. Actual_vs_Predicted.png
Shows the flattened comparison between actual egg prices and model
predictions over the test period. This plot highlights how closely
the GRU model tracks real price movements.

---

### 2. next_7_day_forecast.png
Displays the 7-day ahead forecast for the first test sample.
This visualization demonstrates the model’s short-term forecasting
capability.

---

### 3. correlation_heatmap.png
Pearson correlation heatmap illustrating the relationship between
egg prices and all input features. This helps interpret feature
importance and economic relevance.

---

### 4. metrics.txt
Contains numerical evaluation metrics:
- MAE
- RMSE
- MAPE
- R² score

These metrics quantify prediction accuracy on the test dataset.
